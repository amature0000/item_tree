# item-tree webpage
This project displays an item tree to improve visibility of item structures.

# Maintenance
Install a Python 3 environment and run the following command:
```bash
python3 makedata.py
```
You can then follow the prompts to update the item dictionary. Item quantities should be specified in the format `Item#Quantity`.

You can also directly edit the updated item dictionary in `data.json`. modifying `data.js` will not be reflected in future update actions.
